A COMPUTER SYSTEM


1.Monitor

A computer monitor is an output device that displays visual information generated by a computer. It allows users
to interact with the computer's interface and see the results of their actions. Monitors come in various sizes
and technologies, with LCD (Liquid Crystal Display) and LED (Light Emitting Diode) being the most common.



2.System Unit

A Central Processing Unit (CPU) is the "brain" of a computer, responsible
for executing instructions from software and controlling other components.
It's a fundamental hardware component in any computing system.


3.Mouse

A mouse, in a computing context, is a handheld hardware input device used to control the movement of a cursor on a computer screen, allowing users to interact with graphical user interfaces (GUIs) by selecting, clicking, and dragging items.


4.Keyboard

A keyboard is an input device used to enter characters and commands into a computer or other electronic device.
It functions as a set of buttons or keys that, when pressed, send electrical signals to the device, which then
interprets the signals and displays the corresponding characters or performs the desired action.


5.Flash Drive

A flash drive, also known as a USB drive, pen drive, or thumb drive, is a portable storage device that uses flash
memory to store and transfer data. It connects to a computer or other device via a USB port and is commonly used
for data storage, backup, and file transfer.



Printer

A printer is a device that produces a physical representation of digital information, typically on paper, but
also on other materials like film or 3D objects. Printers are essential for creating hard copies of documents,
images, and other data, and they come in various types with different technologies and capabilities.



Web Camera

A webcam is a digital video camera that connects to a computer, allowing real-time video and audio transmission
typically connect via USB and can be built into devices or be external peripherals.



Speaker

A speaker is a device that converts electrical signals into audible sound. It's a crucial output device for
computers and other audio systems, allowing users to hear audio. Speakers work by using a diaphragm or cone to
vibrate and create sound waves from electrical signals.
